classNames
Функция для объединения CSS-классов в одну строку, игнорируя ложные значения и объекты с ложными значениями.
Синтаксис
classNames(...params)
Параметры
...params — переменное количество аргументов, которые могут быть строками, числами, объектами или другими значениями.
Возвращаемое значение
string — строка, содержащая список CSS-классов, разделённых пробелом.
Описание
Функция принимает произвольное количество аргументов, которые могут быть:
Строками или примитивами — добавляются, если являются "истинными" (truthy).
Объектами — рассматриваются как ключ: булевое_значение. Только ключи с истинными значениями добавляются.
Falsy значениями (null, undefined, false, 0, "") — игнорируются.
Примеры
classNames("btn", "primary");                           // → "btn primary"
classNames("btn", "", "active");                       // → "btn active"
classNames({ active: true, hidden: false });           // → "active"
classNames("btn", { active: true }, null, "primary");  // → "btn active primary"
